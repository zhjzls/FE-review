对象

无序属性的集合，其属性可以包含基本值、对象或者函数。

每个对象都是基于一个引用类型创建的。

  1. 理解对象

    创建对象：

    • 创建Object实例 var obj = new Object()
    • 对象字面量方式 var obj = {}

  2. 对象属性类型： ECMAScript 中有两种属性：数据属性 和 访问器属性

      数据属性：
    • [[configurable]] 决定能否通过delete删除属性，从而重新定义属性， 能否修改属性的特性。 默认为true。
    • [[Enumerable]] 枚举属性; 表示能否通过 for-in 循环返回属性。 默认为true。
    • [[Writeable]] 表示能否修改属性的值。 默认为true。
    • [[value]] 读写当前属性的数据值。 默认为true。

      • 修改属性的特性;

      Object.defineProperty(< 属性所在的对象>, < 属性名>, < 描述符对象(数据属性构成的对象)>)

      访问器属性：
    • [[Configurable]]
    • [[Enumerable]]
    • [[Get]] 读取属性时调用函数。 默认undefined
    • [[Set]] 写入属性时调用函数。 默认为undefined

      • 不一定非要同事指定getter 和 setter。 只指定 getter 时，代表 属性不能写入。 只指定setter函数的属性，会导致该属性不能读其属性值。

    可以通过Object.defineProperties()同时定义多个属性。
  3. 读取属性的特性 Object.getOwnPropertyDescriptor()
